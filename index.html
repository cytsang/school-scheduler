📖 Template Data Entry Shorthand Code Manual

×

👤 Sheet 1: Staff_Roster

  • Discipline: Enter Y for Discipline Master/Mistress (+1.0h credit).
  • Chinese / English / Maths: Enter Y to flag subject specialization.
  • Ratio: Enter Y to apply the target invigilation multiplier to this colleague.

📅 Sheet 2: Availability

  • Tokens: A (08:00-10:30), B (10:30-13:00), C (13:00-16:00).
  • Enter these letters under the specific Date column to mark a teacher as blocked.

⚙️ Sheet 3: Session_Rules

  • Target_Type: Session_ID, Subject, or Form.
  • Target_Value: The specific ID or name (e.g., "CES").
  • Constraint_Condition: Use ONLY_CHINESE_TEACHERS, ONLY_ENGLISH_TEACHERS, or ONLY_MATHS_TEACHERS.

📊 Sheet 4: Exam_Master

  • Structure: Columns for Session_ID, Date, Start Time, End Time, Form, Subject, Venue, and Posts Required.
  • Posts Required: String code where C = Chief, I = Invigilator, B = Beginner, E = Ender.

📌 Sheet 5: Pre_Assignments

  • Purpose: Pin specific teachers to sessions manually.
  • Columns: Session_ID, Post (e.g., "Chief"), and StaffName (exact match from Staff_Roster).

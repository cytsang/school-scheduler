📖 Template Data Entry & Shorthand Code Manual

Reference guidelines for offline Excel spreadsheet configuration rules

👤 Sheet 1: Staff_Roster

  • Discipline Column: Enter Y to flag a Discipline Master/Mistress. Grants a +1.0h baseline offset credit automatically.
  • Chinese / English / Maths Columns: Enter Y to register subject specializations.

📅 Sheet 2: Availability

Configures leaves using exact structural math intersection envelopes:

  • Date Headers: Enter as a 4-digit short text string (e.g., 0608 or 0612).
  • Cell Value Block Tokens:
    • A = Unavailable from 08:00 - 10:30.
    • B = Unavailable from 10:30 - 13:00.
    • C = Unavailable from 13:00 - 16:00.
    • Overlap Protection Engine: Typing A will block an exam row running from 10:15 - 11:45 because it intercepts the 08:00-10:30 boundary window mathematically.

🛠️ Sheet 3: Session_Rules

  • Target_Type: Options include Subject, Form, or Session_ID.
  • Constraint_Condition: Codes include ONLY_CHINESE_TEACHERS, ONLY_ENGLISH_TEACHERS, ONLY_MATHS_TEACHERS, and NO_BEGINNER_POST.

💡 Sheet 4: Exam_Master

  • Venue Column = OC: Auto-explodes into 5 independent classroom rows.
  • Form Column: Combined flags J maps Forms 1 & 2, while S maps Forms 4 & 5.
  • Required Tokens: C (Chief), I (Invigilator), B (Beginner), E (Ender).

📌 Sheet 5: Pre_Assignments

  • Post Shorthand: Supports short text tags like C, Chief, I, or Invig mapped sequentially.
  • Blank Timestamps: Auto-inherits parent timing blocks from Exam_Master and duplicates across both shifts for split sessions.
